用 Octopress 得先学 Markdown

Jan 19th, 2013 | Comments

关于Markdown

Markdown是一种to HTML的语言，就如同coffeescript to javascript、less to css一样。 当然这种语言存在必然有其存在的意义，否然直接写html好了啦。 Markdown官方解释是其用途是：用來作為一種網路內容的寫作用語言。 写作更多的是处理文字的排版等，那么直接用HTML去写确实大材小用，Markdown便可用来方便编写以及提升源文件的可阅读性。

语法:

  • HTML 语法

首先，Markdown是支持直接写HTML的，就如同在less文件中也可以直接写css。

testbbb&>< (看，<、>、&等不需要转义使用了哦~)

  • 段落和換行

Markdown中一个以上的空行会分隔出段落，下面fisrtP会在一个段落里，但是我们会发现中间有个空格。 那是因为浏览器会把p标签内的文本换行转化为一个空格。

firstPfirstPfirstPfirstPfirstPfirstPfirstP firstPfirstPfirstP

secondPsecondPsecondP

  • 标题

很容易理解，这里只列出最简单的一种写法。

This is an H1

This is an H3

This is an H6
  • 引言

就是像回复邮件中引用的那种情形。

This is the first level of quoting.

This is nested blockquote.
  • 列表

有序列表的数值改动是无用的，而且貌似只能从1开始。

1.  Bird
2.  McHale
3.  Parish
    *   Red
    *   Green
    *   Blue
  • 代码块

4个空格或一个tab缩进即代表代码块，这里只是处理成了pre元素，octopress提供了更好的代码展示插件。

This is a normal paragraph:

This is a code block.
  • 分割线
  • 链接

This is an example inline link.

  • 文字修饰

single asterisks

double asterisks

printf()

A backtick-delimited string in a code span: `foo`

  • 图片

octopress也提供了更好的图片插件。

  • 转义字符

用反斜杠来转义。

*literal asterisks*



上面是对markdown最常用，最简单的一些语法进行了介绍，后面还会介绍下octopress编写方面的一些插件。
